You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /distance?lat1=40.7128&lon1=-74.0060&lat2=51.5074&lon2=-0.1278 - measures the distance between two points given the lat and long of both locations
  • /calculatebmi?weight=70&height=1.75 - calculates your BMI, 70 and 1.75 are used as temp values
